JDBC (Java Database Connectivity) - средство интегрирования SQL; java API для работы с различними СУБД.

-----------------------------------------------------------------------------------------------------------------------------
Соединение с базой данных описывается классом, реализующим интерфейс java.sql.Connection. Имея соединение с базой данных, можно создавать
объекты типа Statement, служащие для исполнения запросов к базе данных на языке SQL.

Существуют следующие виды типов Statement, различающихся по назначению:
1. java.sql.Statement — Statement общего назначения;
2. java.sql.PreparedStatement — Statement, служащий для выполнения запросов, содержащих подставляемые параметры 
   (обозначаются символом '?' в теле запроса);
3. java.sql.CallableStatement — Statement, предназначенный для вызова хранимых процедур.

Интерфейс java.sql.ResultSet позволяет обрабатывать результаты запроса.

-----------------------------------------------------------------------------------------------------------------------------
  Пример: 
Мы делаем запрос в БД, нам нужно прочитать 1000 строк. Делаем select и получаем ResultSet.
  Что в этот момент происходит под капотом? 
В действительности, все 1000 строк не загружается в оперативную память из БД, загрузка происходит порциями, каждая порция, - это 
так называемый RowSet. С этим RowSet мы и работаем в каждый момент времени. Размер RowSet мы можем установить используя метод 
statement'a - setFetchSize(int). По умолчанию FetchSize == 10. Новый RowSet подгружается так: ResultSet.next().
