JDBC (Java Database Connectivity) - средство интегрирования SQL; java API для работы с sql 


В программном обеспечении data access object (DAO) — это объект, который предоставляет абстрактный интерфейс к какому-либо типу базы данных или механизму хранения.

Пример: 
 Мы делаем запрос в БД, нам нужно прочитать 1000 строк. 
 Делаем select и получаем ResultSet
  в действительности все 1000 строк не загружается в оперативную память из БД, загрузка происходит порциями, 
  каждая порция это так называемый RowSet. 
 С этим RowSet мы и работаем в каждый момент времени.
 Хоть явно его в коде мы и не видим (кажется что существует только ResultSet).
 Так вот, размер RowSet мы можем установить используя метод statement'a - setFetchSize(int) - вот второй термин. 
 Новый RowSet подгружается неявно - ResultSet.next() По умолчанию FetchSize = 10
